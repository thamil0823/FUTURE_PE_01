AI YouTube Video Creation Using Prompt Chains

This project demonstrates how prompt engineering and AI tools can be used
to create a complete YouTube video from start to finish.

Niche:
Web3 & Crypto Education for Beginners

Topic:
What REALLY Happens When You Buy Crypto?

Objective:
To use a structured prompt chain approach to generate a YouTube video idea,
title, script, thumbnail concepts, and SEO content using AI.

Prompt Chain Workflow:
1. Topic validation
2. Title generation
3. Script generation
4. Thumbnail idea generation
5. SEO optimization
